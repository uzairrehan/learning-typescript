initialize a new package.json file:
npm init -y


install TypeScript locally in your project:
npm install typescript --save-dev


create a tsconfig.json file:
npx tsc --init


Create your TypeScript file:
name.ts


To compile TypeScript locally:
npx tsc name.ts


run the compiled JavaScript file using Node.js:
node app.js


real-time updates in TypeScript:
npx tsc --watch


Run TypeScript Directly with ts-node:
npm install ts-node --save-dev

run your TypeScript file directly:
npx ts-node app.ts
